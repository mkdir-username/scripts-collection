# Tools

Директория для Python утилит и вспомогательных инструментов.

## Структура версий

- `v1.0.0/` - Начальная версия утилит
- `v1.1.0/` - Минорные обновления и новые функции
- `v2.0.0/` - Мажорное обновление с новым API
- `current/` - Символическая ссылка на актуальную версию (v2.0.0)

## Правила именования

Все файлы должны следовать паттерну:
`{name}_v{major}.{minor}.{patch}.{ext}`

Примеры:
- `file_processor_v2.0.0.py`
- `data_converter_v1.1.0.py`
- `api_client_v2.0.0_refactored.py`

## Категории инструментов

- **Обработка файлов** - утилиты для работы с файлами
- **Конвертеры данных** - преобразование между форматами
- **API клиенты** - взаимодействие с внешними сервисами
- **Анализаторы** - инструменты анализа кода и данных

## Использование

```bash
# Использовать актуальную версию
cd /Users/username/Scripts/tools/current/

# Использовать конкретную версию
cd /Users/username/Scripts/tools/v1.0.0/
```

## Зависимости

См. файл `requirements.txt` в каждой версии для установки зависимостей:
```bash
pip install -r requirements.txt
```
